# Descargar Processing

Processing es un IDE y biblioteca de gráficos de Java. *making it easy to learn Java without having to worry about complex configuration*. Se lo puede descargar [aquí](https://processing.org/download/) con instrucciones de instalación simples.

# Básicos

Aquí vamos a dicutir algos de los principios básicos de programar. Vamos a usar enteros, y aprender sobre declaraciones de if/else.

---

## La Forma Básica

Este es la forma básica de el código en Processing:

```java
void setup(){ //es difícil traducir, pero creo que la palabra en español es "fundar" o "establecer"
  //El código aquí se ejecuta solo una vez
}

void draw(){ //en español, "dibujar"
  //El código aquí se ejecuta y se repite, después de `void setup()`
}
```

La códiga de arriba tiene comentarios (después de '//') para explicar lo que hace el código, cada seccíon se llama un "method" (en español, método). 

Miremos una programa pequeña, 


```java
//declara variables después de methods
int variable = 10;

void setup(){
//no hay nada para ejecutar en setup aquí
}

void draw(){
    //print (en español imprimir o escribir) lo que la variable se establece, en este caso 10, repetidamente
     println(variable);
}
```

Hay algunas cosas importante para observar en el código de arriba:

* Hay una ';' despues de comandos
   * *This is because in Java*, se podría escribir todo de una programa en una línea, e.g `void draw(){println(variable);}` La computadora no "vea" *when we end a line*, es solo para que la gente lo pueda leer, la computadora solo vea el final de una línea cuando hay ';'s
* No hay una ';' después de `void draw()` o `void setup()`
    * *This is because* hay '{'s después de esos. No son comandos, pero seccíones de código, entonce en vez de una ';' después de eso, los '{}'s tienen el código.
* El `int variable = 10;` afuera de los methods
    * Este 'declares' (en español 'declara') los variables, le dice los variables que se usan más adelante en el código a la computadora. Tiene 'int' a la principio porque el variable es una integer (en español, entero).

Ya sabes la forma básica, hablemos sobre declaraciones de if/else.

## Declaraciones de If/Else

If you are familiar with any other programming language, this should be very easy for you, the basic setup of if and else statements is:

```java

if(//condition){
    //run this code
}

```

So lets look at an example (we will also work with real graphics, not just the console):

```java
void setup(){
  size(500, 500); //set the size of the graphics window to 500 * 500
}

void draw(){  
  if(mousePressed){ //if the mouse is pressed...
    background(0);
    //...draw a rectangle at the coordinates (250, 250), with a width and height of 100-
    rect(250, 250, 100, 100);  
  }
  else{ //if the mouse is not pressed...
    //...draw the background to cover up the previous rectangle
    background(0);
  }
}
```

In setup we use the  `size` function, which sets the size of the graphics window.
In draw, we use an if statement. If the condition between the ()s is true, then it runs the code between the {}s, if it is not true, it moves onto the next section of code, between the {}s after "else". Try replacing the x and y coordinates of the rectangle with `mouseX` and `mouseY` and see what happens.
